# Anthropic 学习仓库

这是给 Allen 准备的 Anthropic 学习目录，按学习价值拆成 5 个模块：

1. Engineering
2. Research
3. Blog & News
4. Learn & Tutorials
5. Policy & Commitments

## 建议学习顺序

- 第一轮：先看 `01-Engineering`
- 第二轮：补 `02-Research`
- 第三轮：按兴趣穿插 `03-Blog-News`
- 第四轮：需要实操时看 `04-Learn-Tutorials`
- 第五轮：想补底层安全哲学时看 `05-Policy-Commitments`

## 每篇文件怎么用

每个文件都预留了：
- 文章链接
- 为什么值得读
- 阅读关注点
- 你的笔记区

建议你阅读时只补三件事：
1. 这篇文章最重要的 3 个观点
2. 哪些内容能迁移到你自己的 agent / workflow
3. 你不同意或仍然存疑的地方
