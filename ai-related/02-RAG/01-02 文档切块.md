# Step 1：文档切块（Chunking）

因为：

LLM context 有长度限制。

所以不能直接：

```text
整个项目 → 扔给 GPT
```

必须：

## 切块（chunk）

例如：

```text
订单模块 README
```

被切成：

```text
chunk1
chunk2
chunk3
...
```

每块可能：

- 256 tokens
    
- 512 tokens
    
- 1000 tokens
    

---

## 为什么切块很重要

因为：

### 切太小：

上下文丢失。

### 切太大：

召回不精准。

这是 RAG 最核心调参之一。
