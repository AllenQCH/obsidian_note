### 如果你只想先看最重要的结论

#### 结论 1：你的目标不是“算法科学家”
更合理的目标是：

- AI Application Engineer
- AI Agent Engineer
- AI Backend / AI Infra

#### 结论 2：学习主线不要乱
最推荐的顺序是：

```text
方向判断
-> Python 工程能力
-> Prompt / RAG
-> Agent / Workflow
-> MCP / Tool Calling
-> 业务场景项目
```

#### 结论 3：要靠项目转型，不要靠囤知识
优先做这类项目：

- 企业知识库 / RAG
- AI SQL Agent
- 业务排障 Agent
- SOP / 工作流 Agent
