**1. 模块（module）是什么**
模块本质就是：一个 Python 文件
例如：math.py  、typing.py  、random.py都叫模块。类似 Java 的：package + class
```
import math
就是导入 math 模块
math.sqrt(4)调用里面的方法。

```
**typing 是什么**
`typing` 是 Python 官方提供的一个模块。专门用来：做类型标注
里面有：Any 、List 、Dict  、Optional  、Union；这些“类型工具”。
**2. 类型（type）是什么**
类型本质：数据的分类
例如：1 -> int  ；"abc" -> str  ；True -> bool
**Any 是什么**
`Any` 表示：任意类型都可以

**为什么需要 typing**
因为 Python 原本是动态类型：但大型工程里不好维护。所以后来加入：类型提示（Type Hint）


### 命名规范：
模块名 -> 小写
fastapi  
typing  
json  
datetime
类名 -> 大驼峰（PascalCase）
FastAPI  
BaseModel  
HTTPException  
UserService

Python 常见命名规范，你可以这样记：
可以发现：python是没有驼峰命名的，只有大驼峰命名

| 类型    | 规范   | 示例         |
| ----- | ---- | ---------- |
| 模块/文件 | 小写   | fastapi.py |
| 包/目录  | 小写   | routers    |
| 类     | 大驼峰  | FastAPI    |
| 函数    | 下划线  | get_user   |
| 变量    | 下划线  | user_name  |
| 常量    | 全大写  | MAX_SIZE   |
| 私有变量  | 单下划线 | _token     |
| 强私有   | 双下划线 | __name     |
