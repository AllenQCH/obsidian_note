# 00-top firms总览

## 摘要

这组笔记的目标，不是做“公司八卦合集”，而是帮我建立一个 **AI 顶级公司观察框架**：
- 谁更强在模型
- 谁更强在 research
- 谁更强在 agent / tool use / eval
- 谁更强在 open ecosystem
- 谁更值得长期跟踪

如果我的目标是继续往 **AI Agent 工程 / LLM 工程 / 多 agent 系统** 方向走，那么这些公司就是最值得持续追踪的一批样本。

## 核心内容

### 我为什么要看 top firms

看 top firms，不只是为了知道“谁最火”。
更重要的是：
1. 看行业主线往哪走
2. 看哪些能力正在从研究变成产品
3. 看哪些工程模式会被大规模复用
4. 看哪些公司最值得长期学习它的方法论

### 一个最实用的判断框架

| 公司 | 最强印象 | 我最该看什么 |
|---|---|---|
| Google DeepMind | research 深、强化学习强、科学 AI 强 | planning、agent eval、长任务、Gemini 体系 |
| OpenAI | 产品化快、开发者生态强 | ChatGPT、API、Agents SDK、operator/assistants 路线 |
| Anthropic | agent engineering 与安全边界强 | Claude、tool use、context engineering、reliability |
| Meta AI | 开源生态推动力强 | Llama、开源模型路线、多模态与 infra 扩散 |
| xAI | 产品传播力强、实时性与分发独特 | Grok、实时信息流、消费级 AI 产品打法 |

### 我后面追这些公司的重点主题

1. **模型能力**：推理、多模态、长上下文、速度/成本
2. **Agent 能力**：planning、memory、tool use、long horizon tasks
3. **工程体系**：SDK、workflow、runtime、eval、observability
4. **安全与可靠性**：guardrails、sandbox、policy、alignment
5. **生态位**：API、开源、应用分发、企业落地、研究影响力
