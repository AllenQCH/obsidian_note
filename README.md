# Obsidian Notes

个人 Obsidian 笔记仓库。

## 目录

- `Codex工作台/`：Codex 工作日志、项目记忆、流程规范和同步配置
- `ai-related/`：AI、Agent、MCP、CLI、自动化网页操作等相关笔记
- `吴恩达视频/`：吴恩达课程和视频相关笔记
- `项目相关/`：项目实践相关笔记
- `未命名/`：暂未归类的笔记

## 同步说明

本仓库保留笔记内容和必要图片，不提交 Obsidian 本地配置目录 `.obsidian/` 和系统文件。

当前本地仓库通过 SSH alias 同步到：

```text

```
