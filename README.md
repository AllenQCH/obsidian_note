# Obsidian Note

This repository is prepared as an Obsidian vault for personal Codex work notes.

## Structure

- `Codex工作台/工作日志/`: session summaries and daily work records.
- `Codex工作台/项目记忆/`: reusable project context extracted from Codex memories.
- `Codex工作台/流程规范/`: durable workflow rules and operating conventions.
- `Codex工作台/同步配置/`: GitHub sync notes and local setup instructions.
