
```python
app_an_py310/
│
├── main.py                            # 项目入口
│
├── dependencies.py                    # 公共依赖（token校验等）
│
├── __init__.py                        # 标记当前目录是 Python 包
│
├── internal/                          # 内部模块
│   │
│   ├── admin.py                       # 管理员相关接口
│   │
│   └── __init__.py
│
└── routers/                           # 业务路由层
    │
    ├── items.py                       # items 相关接口
    │
    ├── users.py                       # users 相关接口
    │
    └── __init__.py
```
这个项目的职责划分：
main.py
```python
负责：
- 创建 FastAPI 应用
- 注册所有 router
- 配置全局依赖
- 配置 prefix/tags
类似：
- SpringBoot 启动类
```
dependencies.py
```python
负责：
- token 校验
- Header 校验
- 公共 Depends
```
routers/
```python
负责：
普通业务接口
类似：
- Controller
```
internal/
```python
负责：
内部模块 / 管理模块
这里：
admin.py
通常表示：
- 管理员接口
- 内部接口
- 非公开功能
```
