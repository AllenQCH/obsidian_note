# AI英语学习群-生词库

## 摘要

这是 AI 英语学习群｜Easy Daily 的长期生词库。后续每日英语文章和 10:00 记忆曲线复习任务，应优先读取这里的词，再结合群聊里最新提交的生词。

## 使用规则

- 学员新提交的生词，优先追加到 `## 按日期记录`。
- 10:00 复习任务优先选择最近 1-7 天的词，并做 D1 / D3 / D7 分层。
- 08:30 新文章可自然复现 2-4 个近期词，不要硬塞。
- 同一个词可保留多次语境，但复习时要去重归并。

## 高频复习池

- traffic：技术语境中常指“流量 / 请求量”，不是交通。
- asynchronous processing：异步处理，常见于 task queue、worker、background job。
- duplicate / idempotency：重复消息与幂等性经常一起出现。
- coupon：在后端例子里常指优惠券功能、优惠券服务或优惠券数据。
- schema / migration / compatible：数据库迁移和 API 兼容场景常一起出现。
- latency / throughput / metrics：性能监控三件套。
