# 2026-04-30 Codex 默认工作区配置

## 背景

希望 Codex 默认打开目录改为：

```text
/Users/heytea/Documents/HeyTea/codex-workspace
```

该目录已声明为 Miscellaneous workspace，适合放非项目任务，例如写作、研究、面试准备、临时分析和个人规划。

## 本次变更

修改启动脚本：

```text
/Users/heytea/Desktop/Codex-Quick-Start.command
```

把原来的默认目录 `$HOME` 改为：

```zsh
default_codex_dir="/Users/heytea/Documents/HeyTea/codex-workspace"

cd "$default_codex_dir" || exit 1
codex -C "$default_codex_dir" -a never -s workspace-write "$@"
```

## 验证

- `zsh -n /Users/heytea/Desktop/Codex-Quick-Start.command`
- 检查目标目录存在。
- 检查目标目录下存在 `AGENTS.md`。

## 限制

该修改影响桌面的 `Codex-Quick-Start.command` 启动入口。若从 CodexBar 菜单栏 App 打开 Codex，可能还有独立的启动目录设置。
